SKILL

できること

HTML/CSS

スマホで見ても表示崩れのないレスポンシブ対応も可能です。Visual Studio Codeを使用しています。

WordPress

WordPress製の店舗HP・企業HP・メディアサイトなど、Webサイトを0から構築することが可能です。

JavaScript

お問い合わせフォームや自動スクロールなど動きのあるWebサイトを作る事が可能です。

Adobe Photoshop

Webサイトに必要不可欠なクリエイティブバナーの作成、写真の編集などが可能です。

jQuery

Webサイトにフェードイン・フェードアウトなどリッチな動きをつけることが可能です。

Adobe XD

Webサイトのデザインやワイヤーフレームを作ることが可能です。

STUDIO

レスポンシブ可能なWebサイトやLPをノーコードで作成することが可能です。

Adobe Illustrator

ロゴ、名刺、資料、パンフレット等の作成が可能です。当サイトのロゴもillustratorにて一から作成しました。

ABOUT

私について

清水 結唯

Shimizu Yui

はじめまして、mugicha designの清水と申します。
北海道を拠点にWEBデザインやWeb制作をしております。 2018年から6年間、事務職として勤務し、業務の効率化や細部への注意力を培いました。その傍ら、以前から興味を持っていたWebサイト制作を学び、2024年より本格的に活動を開始いたしました。 現在は、Web制作において、デザイン、コーディング、スマートフォン対応を得意としています。お客様のご要望に寄り添い、低予算でも魅力的で使いやすいWebサイトをご提供します。
まずはお気軽にお問い合わせください。あなたのビジョンを形にするお手伝いをさせていただきます。
